Supprimer votre compte DERCOMIA

Cette page explique comment supprimer définitivement votre compte ainsi que toutes les données personnelles associées à votre profil.

Comment supprimer votre compte

1. Ouvrez l'application DERCOMIA.
2. Connectez-vous à votre compte.
3. Accédez à la page Profil.
4. Appuyez sur Supprimer mon compte.
5. Une fenêtre de confirmation apparaît.
6. Saisissez votre nom dans le champ demandé.
7. Appuyez une seconde fois sur Supprimer mon compte afin de confirmer la suppression définitive.

Données supprimées

Lors de la suppression de votre compte, les éléments suivants sont définitivement supprimés :

  • Compte Firebase Authentication
  • Profil utilisateur
  • Nom et prénom
  • Adresse e-mail
  • Numéro de téléphone
  • Adresse de livraison
  • Favoris
  • Informations de fidélité
  • Toutes les données associées à votre compte

Données conservées

Aucune donnée personnelle n'est conservée après la suppression du compte, sauf si une obligation légale impose une période de conservation.

Suppression définitive

La suppression est irréversible. Une fois votre demande confirmée, votre compte et toutes les données associées ne pourront plus être récupérés.
